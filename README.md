# Analise_de_comentarios
Este projeto realiza a classificação automática de comentários em três categorias:  
- Elogios
- Críticas
- Outros  
A classificação é feita com base em palavras‑chave definidas manualmente e aplicadas a um conjunto de respostas armazenado em um DataFrame do pandas. Após a classificação, o projeto exporta cada categoria para um arquivo Excel independente.
